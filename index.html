COMPLETIONISM.CO.UK
Website for tracking collections in the game World of Warcraft. Still a work in progress,
but I've received a lot of positive feedback from the community so far, and will be
continuing development! Written in PHP and Javascript, and uses the developer's API
to import information about players' in-game collections.

You can try syncing a character by clicking 'Link' and entering the following information:
Category - Mounts / Region - EU / Realm - Sylvanas / Character - Giblut
Now click the big 'Mounts' button to view the collection.
